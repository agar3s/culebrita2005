# Culebrita con graficos vectoriales

Juego de batalla entre 2 jugadores, cada jugador controla una culebra que debe ir coleccionando diferentes bonos (power ups), las culebras pueden moverse en cualquier direccion y sus sprites son graficos vectoriales.  

El proyecto es del año 2005 y esta realizado sobre Java 1.4
Recomiendo Eclipse IDE para su ejecución.
